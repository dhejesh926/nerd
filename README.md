# nerd
A tui based nerd glyph picker built using bash supported for linux and macos.

## This is how it looks
<img width="1325" height="709" alt="screenshot_20260519_110218" src="https://github.com/user-attachments/assets/4af94525-ae31-4ee8-9662-4c28225c3c64" />
Just type the name of the nerd glyph you are looking for, and press return key on the glyph, the glyph will be copied onto your clipboard.

## Prerequisites
fzf(fuzzy finder),
 wl-copy(for wayland users),
 xclip(for x11 users),
 any nerd font as your terminal font.
